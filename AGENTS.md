# SAP CAP and Fiori Rules

- Use cds-mcp to search CDS entities, services and models before reading .cds files.
- Use cds-mcp search_docs before modifying CAP models or using cds APIs.
- Use fiori-mcp for SAP Fiori Elements generation and modifications.
- Use ui5-mcp for UI5 documentation and code generation.
- Generate UUID keys for all entities.
- Use SAP Fiori Elements whenever possible for CRUD applications.
